# 데이터 액세스 계층

데이터 액세스 계층은 서비스 계층에서 비즈니스 로직을 통해 처리된 데이터를 **데이터베이스에 저장**하고,  
비즈니스 로직을 처리하기 위해 데이터베이스에서 **데이터를 조회**하는 역할을 한다.

<br>

Spring에서 사용하는 데이터베이스 액세스 기술에는  
mybatis, Spring JDBC, Spring Data JDBC, JPA, Spring Data JPA 등이 있다.
