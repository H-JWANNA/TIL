## 메서드란?

### **어떤 기능을 수행하기 위한 코드들을 묶어놓은 것**

<br>

데이터를 입력받아 메서드 내부의 기능을 수행하고 결과를 출력한다.

<br>

```java
Public void main(int A) {
	B
}
```

> Public : 접근제어자  
> void : 반환 타입  
> main : 메소드 이름  
> int : 매개변수의 타입  
> A : 매개변수의 이름  
> B : 메서드 바디
