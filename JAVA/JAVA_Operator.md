# 연산자

## 연산자의 종류 및 우선 순위

| 우선순위 | 연산자 | 내용 |
|:------:|:---:|:---:|
| 1 | (), [] | 괄호 / 대괄호 |
| 2 | !, ~, ++, -- | 부정 / 증감 연산자 |
| 3 | *, /, % | 곰셈 / 나눗셈 연산자 |
| 4 | <, <=, >, >= | 대소 비교 연산자 |
| 5 | && | AND 연산자 |
| 6 | \|\| | OR 연산자 |
| 7 | ? : | 조건 연산자 |
| 8 | =, +=, -=, /=, %= | 대입 / 할당 연산자 |


- 삼항 연산자 : ```int a = (조건문) ? 참일때 : 거짓일때;```
